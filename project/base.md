# 10. Минимальная весовая триангуляция (Minimum Weight Triangulation)

__Цель__: Исследовать алгоритмы построения триангуляции и разработать приближённые методы для минимизации суммарного веса.

__Постановка задачи__: Дано множество точек на плоскости. Требуется построить триангуляцию с минимальной суммарной длиной рёбер. Задача является вычислительно сложной, поэтому необходимо рассмотреть приближённые алгоритмы и эвристики. Требуется предложить улучшение одного из подходов.

__Данные__: Случайные и структурированные наборы точек (кластеризованные, равномерные). Графовые структуры для навигационного меша различного вида.

__Экспериментальная часть__: Сравнить различные методы (например, Delaunay и его модификации) по суммарному весу. Исследовать влияние распределения точек. Предложить и обосновать собственную эвристику. Выполнить сравнение качества навигационного меша, построенного триангуляцией Делоне и эвристической минимальной весовой триангуляцией, на метриках: длина пути, количество поворотов, время построения

__Ожидаемый результат__: Программа триангуляции с визуализацией и анализ качества решений. Требуется визуализация результатов.
